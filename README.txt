This project implements an GIF gallery with random intervaled gifs search
In the search bar write a subject and time example: "cat 3", then every 3 seconds a random cat gif will appear in the gallery.

Create a GIF API key and put in the server app.js file.
Frontend is in React
Backend Node.js

1. Go to "fclient" folder and run:
	npm install
	npm run-script build
	
2. Go to "server" folder and run:
	npm install
	npm start

3. In browser enter "localhost:9000"
